# Word Relation Analogy

## How to run this project

### Prerequisite

1. Java 11
2. Maven
3. Terminal
4. IDE
5. Browser

### Build to project from terminal

1. Maven clean
   ```shell
   mvn clean
   ``` 
2. Maven install
   ```shell
   mvn install
   ``` 
3. Run Project
   ```shell
   mvn spring-boot:run
   ``` 
4. Once project is running, browse the swagger URL for APIs detail
   http://localhost:8080/swagger-ui/index.html
